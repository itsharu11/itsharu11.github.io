What is Process Hacker

Process Hacker is a free, open source, graphical tool for managing 32-bit and 64-bit Microsoft Windows processes, services, threads, memory, handles, modules, Security Access Tokens (SATs) and network connections

It is a wonderful tool for analyzing and combating malware, understanding low-level details of the Windows operating system, troubleshooting, and experimenting with Windows in ways which Microsoft never intended.

Install and Configuration

Download binaries and source from http://processhacker.sourceforge.net.

Make sure to install the latest version of Microsoft’s "Debugging Tools for Windows (http://msdn.microsoft.com/windows/hardware/), and then in Process Hacker, pull down the Hacker menu > Options > Symbols tab > confirm that you have a valid path to DBGHELP.DLL
